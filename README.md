# Abrar Bari Portfolio

Personal portfolio for Abrar Bari, built with React, TypeScript, and Vite.

## Tech Stack

- React 18
- TypeScript
- Vite
- CSS

## Local Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run typecheck
npm run lint
npm run build
```

## Deployment

This is a static Vite app. Use:

- Build command: `npm run build`
- Output directory: `dist`

The public resume and portrait assets are stored in `public/`. Local design source folders are ignored by git.
